# Playground Functions

Este projeto tem como objetivo praticar os conceitos básicos da linguagem JavaScript, como trabalhar com arrays, objetos, funções, operações aritméticas e a sintaxe da linguagem em geral.


O projeto consiste em implementar 13 funções, cada uma com um desafio diferente. As funções foram implementadas da forma mais simples possível, sem usar conceitos mais avançados do JavaScript. O objetivo é praticar a lógica de programação, não necessariamente a sintaxe ou as funcionalidades mais complexas da linguagem.


Aqui estão alguns dos conceitos abordados no projeto:

Variáveis e tipos primitivos;

Operadores lógicos, aritméticos e de atribuição;

Estruturas condicionais (if/else);

Loops (for, while);

Arrays;

Objetos;

Funções.
